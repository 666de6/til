# Day 51 - mac system hotkey

* `cmd + ,` -> open app setting
* `cmd + q` -> quit app
* `cmd + ctrl + q` -> lock screen
* `cmd + h` -> hide window
* `cmd + m` -> minimize window into sidebar
* `cmd` + ` -> switch between windows within an app
* `cmd + ctrl + f` -> full screen
