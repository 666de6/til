# Day 30 - debug in vscode

`f5` -> start
`shift + f5` -> stop
`shift + cmd + 0` -> restart
`shift + cmd + 9` -> add breakpoint
`cmd + '` -> step over
`shift + cmd + '` -> step into
`f5` -> step out
