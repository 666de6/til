# Day 28 - lazygit

**install**
```bash
brew install lazygit
```
**alias**
```bash
echo "alias lg='lazygit'" >> ~/.bash_profile
```
**git operations**
* stage / unstage single file-> switch by `space`
* stage all files-> `a`
* commit -> `c`
* reset -> `g`
* pull -> `p`
* push -> `shift + p`
* discard single file -> `d`
* discard all files -> `shift + d`
