# Day 02

## row operation

* `0`->move to the beginning
* `^`->move to the beginning(not blank)
* `$`->move to the end
* `g`_->move to the end(not blank)

* `I`->insert to the beginning
* `A`->insert to the end
* `o`->add row(below)
* `O`->add row(above)

* `dd`->delete row
* `yy`->copy row
* `p`->paste
