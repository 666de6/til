# Day 06
## text object
**syntax**
* operator + （internal (i)/ external(a)）+ text object
* visual mode + （internal (i)/ external(a)）+ text object

**text object**
* `w` -> a word
* `(` or `)` -> a pair of ()
* `b` -> a pair of ()

* `[` or `]` -> a pair of []

* `{` or `}` -> a pair of {}
* `B` -> a pair of {} 

* `<` or `>` -> a pair of <>
* `t` -> xml tag
* `s` -> sentence
* `p` -> paragraph

**vim-textobj-arguments**
* `daa` -> delete an argument
* `cia` -> edit an argument

**vim-textobj-entire**
* `dae` -> clear the current page
