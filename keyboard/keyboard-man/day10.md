# Day 10

## number & dot

**number**
`operation + number + action/range`

**dot**
* duplicate the last operation:
  * create / delete / edit
  * everything under insert mode                         

**tips:**
* delete a single word
  * `diw + .`
* add `;`
  * `jA + .`
* replace the same word
  * `search + cw + n + .`
