# Day 56 - edit in obsidian 

* `cmd + e` -> toggle reading view
* `cmd + b` -> bold
* `cmd + i` -> italic
* `cmd + k` -> insert external link
* `cmd + -` -> toggle bullet list
* `cmd + /` -> omit
* `cmd + return` -> create checkbox
* `f2` -> edit file name
* `zm` -> fold all
* `zr` -> unfold all
* `za` -> toggle fold
