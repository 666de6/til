**common prefix** 
* `imp` -> import xxx from 'xxx';
* `imd` -> import { xxx } from 'xxx';
* `fn` -> function
* `anfn` -> arrow function
* `log` -> console.log()
* `iife` -> IIFE function
* `rp` -> return new Promise
