# Day 01 - move & insert

**move**
* `h` -> left
* `l` -> right
* `j` -> top
* `k` -> bottom

**insert**
* `i` -> insert (before)
* `a` -> insert (after)

**back to normal**
* `esc` / `ctrl + [`
