# Day 05 - visual mode = selected + operation

**switch to visual mode**
* `v` -> character
* `V` -> row
* `ctrl + v` -> block

**nav**
* `action` -> hjkl
* `o` -> change focus position
* `gv` -> back to last time 

**exit visual mode**
* `esc`
* `ctrl + [`
* `v`
* `V`

*tips:*
* multiple rows -> `selected + A/I`
* copy/paste -> vim & system
